    12. 6️⃣ LC 106. 从中序与后序遍历序列构造二叉树 🟡

    15. 填充每个节点的 next 指针指向右侧节点。### 🎨 图解思路123456 1 → NULL / \ 2 → 3 → NULL / \ \4→ 5 → 7 → NULL使用层序遍历，连接同层节点### 💻 代码实现1234567891011121314151617def connect(root): if not root: return None queue = deque([root]) while queue: size = len(queue) prev = None for i in range(size): node = queue.popleft() if prev: prev.next = node prev = node if node.left: queue.append(node.left) if node.right: queue.append(node.right) return root### 🧠 记忆口诀> “层序遍历，前连后”

    22. 1️⃣1️⃣ LC 124. 二叉树中的最大路径和 🔴
      1. 题目描述
    23. 找出路径和最大的路径（可以不经过根节点）。### 🎨 图解思路123456789101112 -10 / \ 9 20 / \ 15 7最大路径: 15 → 20 → 7 = 42思路:每个节点可以：1. 只贡献自己（作为路径端点）2. 贡献自己+左子树3. 贡献自己+右子树4. 作为拐点（左+自己+右）### 💻 代码实现12345678910111213141516def maxPathSum(root) -> int: max_sum = float('-inf') def max_gain(node): nonlocal max_sum if not node: return 0 # 左右子树的最大贡献（负数不要） left_gain = max(max_gain(node.left), 0) right_gain = max(max_gain(node.right), 0) # 当前节点作为拐点的路径和 path_sum = node.val + left_gain + right_gain max_sum = max(max_sum, path_sum) # 返回给父节点的贡献（只能选一边） return node.val + max(left_gain, right_gain) max_gain(root) return max_sum### 🧠 记忆口诀> “拐点算全局，贡献选一边”
    24. 1️⃣2️⃣ LC 173. 二叉搜索树迭代器 🟡

    25. 实现二叉搜索树的迭代器。### 💻 代码实现123456789101112131415class BSTIterator: def __init__(self, root): self.stack = [] self._leftmost_inorder(root) def _leftmost_inorder(self, node): while node: self.stack.append(node) node = node.left def next(self) -> int: node = self.stack.pop() if node.right: self._leftmost_inorder(node.right) return node.val def hasNext(self) -> bool: return len(self.stack) > 0### 🧠 记忆口诀> “栈存左链，弹出处理右”
    26. 1️⃣3️⃣ LC 222. 完全二叉树的节点个数 🟢
      1. 题目描述
    27. 统计完全二叉树的节点个数。### 💻 代码实现12345678910111213141516def countNodes(root) -> int: if not root: return 0 left_depth = right_depth = 0 left, right = root, root while left: left_depth += 1 left = left.left while right: right_depth += 1 right = right.right # 满二叉树 if left_depth == right_depth: return 2 ** left_depth - 1 # 递归 return 1 + countNodes(root.left) + countNodes(root.right)### 🧠 记忆口诀> “满树用公式，不满递归数”
    28. 1️⃣4️⃣ LC 236. 二叉树的最近公共祖先 🟡
      1. 题目描述
    29. 找两个节点的最近公共祖先（LCA）。### 🎨 图解思路12345678 / \ 5 1 / \ / \ 6 2 0 8 / \ 7 4LCA(5, 1) = 3LCA(5, 4) = 5### 💻 代码实现12345678910def lowestCommonAncestor(root, p, q): if not root or root == p or root == q: return root left = lowestCommonAncestor(root.left, p, q) right = lowestCommonAncestor(root.right, p, q) # p, q 分布在两边 if left and right: return root # p, q 在同一边 return left if left else right### 🧠 记忆口诀> “左右都有返回根，否则返回有的那边”
