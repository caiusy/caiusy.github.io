比opencv的速度要慢很多, 但是使用起来更加简单, 真的对速度要求很高的话, 一般都会C++和opecv使用. 所以一般情况下, 首先看skimage能否实现, 不行的话再转用opencv  

    
    

```python
    import skimage
    from skimage import io #  IO is a submodule. Submodules need to be imported from the parent module explicitly.
    img = io.imread("1.jpg")
```
