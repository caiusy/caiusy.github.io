## 题目描述

我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？
    
    

```python
    class Solution {
    public:
        int rectCover(int number) {
        if(number <= 0) return 0;
        if(number == 1 || number == 2) return number;
        return rectCover(number - 1) + rectCover(number - 2);
        }
    };
```

  
运行时间：538ms  
占用内存：484k
