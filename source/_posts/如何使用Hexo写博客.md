## 简介

这是一篇示例文章，演示如何使用 Hexo 写博客。

## 步骤

### 1. 创建新文章

```bash
npx hexo new "文章标题"
```

### 2. 编辑文章

在 `source/_posts/` 目录下找到刚创建的 Markdown 文件进行编辑。

### 3. 本地预览

```bash
npx hexo server
```

访问 http://localhost:4000 查看效果。

### 4. 部署

```bash
npx hexo deploy
```

## 总结

使用 Hexo 写博客非常简单！
